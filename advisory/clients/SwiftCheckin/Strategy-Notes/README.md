# Strategy Notes

This folder contains **internal advisor thinking** for SwiftCheckin - strategic analysis, decisions, and deliberations that inform engagement strategy.

## Purpose

| What Goes Here | What Doesn't Go Here |
|----------------|---------------------|
| Internal strategic analysis | Meeting notes (use `Meetings/`) |
| Investment/introduction decisions | Client deliverables (use `Deliverables/`) |
| Engagement strategy decisions | Reference documents (use `Docs/`) |
| AI conversation summaries | Client communications (use `Comms/`) |
| Risk assessments | |

## Naming Convention

**Format:** `YYYY-MM-DD-descriptive-name.md`

**Examples:**
- `2025-12-04-strategic-analysis.md`
- `2025-12-04-investment-gates.md`
- `2025-12-04-engagement-decision.md`

## Document Types

Use frontmatter `type:` field to categorize:

| Type | Purpose |
|------|---------|
| `thinking` | Raw analysis, exploration, deliberation |
| `strategy` | Strategic recommendations and approaches |
| `decision` | Key decisions made about the engagement |
| `conversation` | AI conversation summaries and insights |

## Template

```yaml
---
type: thinking | strategy | decision | conversation
date: YYYY-MM-DD
status: draft | final
---

# Title

## Context
[What prompted this analysis]

## Analysis
[Core thinking/analysis]

## Decisions/Recommendations
[Key takeaways]

## Next Actions
[What to do based on this]
```

## When to Add Documents

- After significant strategic conversations (with client or internally)
- When making engagement decisions (continue, pause, exit)
- When evaluating investment or introduction readiness
- When assessing client health or progress
- After processing inbox files that reveal strategic insights
